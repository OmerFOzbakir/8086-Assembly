# 8086-Assembly
Bu depo, Intel 8086 mimarisi üzerinde x86 Assembly dili kullanarak geliştirdiğim temel algoritmaları ve sistem seviyesi işlemleri içermektedir.

Dil: Assembly (x86 / 8086)

Araçlar: EMU8086 

Düşük seviyeli programlama mantığını pekiştiren çeşitli örnekler yer almaktadır:

Aritmetik İşlemler: 16-bit toplama, eldeli toplama (ADC) ve döngüsel toplama işlemleri.

Dizi ve Mantıksal İşlemler: Dizide maksimum değer bulma, büyük/küçük harf dönüşümü.

Sistem Çağrıları: Ekran temizleme ve ekrana karakter/mesaj yazdırma (INT 21H kullanımı).
